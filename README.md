# voice-of-hadhramaut
مشروع "صوت حضرموت" هو منصة استبيانات إلكترونية متطورة تجمع وتحلل آراء المجتمع الحضرمي حول القضايا المحلية. تهدف لتمكين المواطنين من التعبير عن احتياجاتهم وتطلعاتهم بشكل منظم وآمن، وتقديم بيانات دقيقة لصناع القرار والباحثين. تستخدم أحدث تقنيات الويب والذكاء الاصطناعي المفتوحة المصدر لتحليل الاتجاهات وصياغة تقارير تفاعلية تعكس صوت حضرموت الحقيقي.
